"# ASPNET-DK24TTK3-tathanhtung-vieclam"  
Dự án này là một nền tảng tuyển dụng trực tuyến, kết nối Ứng viên và Nhà tuyển dụng, được quản trị tập trung bởi Admin.

## Công nghệ sử dụng (Tech Stack)
*   **Backend:** .NET 8 Web API
*   **Frontend:** ReactJS (Vite)
*   **Database:** SQL Server

## Cấu trúc thư mục tài liệu
*   `/progress-report`: Chứa các báo cáo tiến độ sau mỗi giai đoạn.
*   `/setup`: Các script, file cấu hình, hướng dẫn cài đặt môi trường.
*   `/src`: Mã nguồn chính của dự án (gồm cả backend và frontend).
*   `/thesis`: Tài liệu báo cáo, slide thuyết trình, bản nháp đặc tả.
